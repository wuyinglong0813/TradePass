# 财源通天 MVP 开发路线

## 阶段 1：基础账号与企业主体

- 微信登录、手机号绑定、当前用户接口。
- 企业信息提交、企业认证状态流转。
- 企业成员、法人、管理员角色模型。

## 阶段 2：我的页面闭环

- 公司信息上传。
- 实名认证和人脸核验接入第三方服务。
- 电子章上传与人工审核。
- 法人给员工开通采购、销售、财务、管理员等授权。

## 阶段 3：供方/需方首页

- 供应商首页：客户销售业绩排名。
- 采购商首页：采购业绩排名。
- 统计维度：总、年、月。
- 首页接口统一返回公司、角色、筛选项和排行数据。

## 阶段 4：订单与排行数据

- 建立订单主表和订单明细表。
- 先支持后台录入或 Excel 导入订单。
- 每日聚合到 `trade_stat_daily`。
- 生成 `ranking_snapshot` 支撑首页快速查询。

## 阶段 5：后台管理

- 企业认证审核。
- 实名核验记录查询。
- 电子章审核。
- 订单导入、查询、修正。
- 排行快照重算。

## 阶段 6：可扩展能力

- 接入对象存储正式上传凭证。
- 接入腾讯云实名核身或等价服务。
- 接入第三方电子签章平台。
- 扩展合同、对账、回款、付款、发票、风控模块。
- 订单量上来后引入 MQ 做实时统计。
